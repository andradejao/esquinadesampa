Sobre o projeto

O Esquina de Sampa é um projeto integrador desenvolvido para conclusão do curso Técnico em Informática no Senac Tatuapé. A aplicação visa divulgar restaurantes de bairro pouco conhecidos em São Paulo, combinando tecnologias de front-end (HTML, CSS, JavaScript) e back-end (Node.js, Express, MySQL) para criar uma plataforma eficiente. O projeto oferece uma experiência multiplataforma, destacando a cultura gastronômica local e aplicando os conhecimentos técnicos adquiridos no curso.
